## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/eltonlazzarin/reactjs-rocketfy-app.git

# Go into the repository
$ cd reactjs-rocketfy-app

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
